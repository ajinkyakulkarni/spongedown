# Spongedown

```bob
                 _____________
 +---------+     \            \        +------------+
 | md+bob  |----->) spongedown )------>| html + svg |
 +---------+     /____________/        +------------+
```

Spongedown converts markdown to html with support for 
[svgbob diagrams](https://github.com/ivanceras/svgbobrus)
into an svg.
